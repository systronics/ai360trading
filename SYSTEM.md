Project OverviewAI360Trading is a fully automated content + trading signal system running on GitHub Actions. It is designed to appear as a high-effort, "manually curated" brand through sophisticated AI scripting.Core Philosophy:Zero-Footprint: Operates 24/7 without a local PC or static IP.Human-Like Quality: Avoids generic AI templates; uses 3D characters (Zeno) and custom-designed data slides.Interconnected Ecosystem: Morning videos, Mid-day shorts, and Nightly reels are interlinked to drive traffic across the platform.2. The Artifact Bridge (Inter-Workflow Communication)Implemented March 27, 2026To solve the "Clean Runner" problem, we use GitHub Artifacts to pass data between scheduled runs.The Producer: daily-videos.yml (7:30 AM) saves the YouTube Video ID to output/analysis_video_id.txt.The Storage: This file is uploaded as a GitHub Artifact named market-analysis-id (Retention: 1 Day).The Consumer: daily-shorts.yml (11:30 AM) downloads this artifact.The Result: Your Mid-day Shorts automatically include a "Watch Full Analysis" link in their descriptions, boosting SEO and cross-platform views.3. Global SEO & Multi-Market StrategyThe system is no longer "India-Only." It targets a worldwide audience (USA, UK, Brazil, India).Data Fetching: generate_analysis.py and generate_shorts.py fetch live data for Nifty 50, S&P 500 (^GSPC), and Bitcoin (BTC-USD).Metadata Logic: Groq generates Hinglish titles (English keywords for Search + Hindi for High CTR).Timezone Sync: All schedules are anchored to IST (Asia/Kolkata) but content is released to catch global "Prime Time" windows.4.
